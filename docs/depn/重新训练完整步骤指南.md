**可选方案**：

## Step 1: 数据格式转换

**目标**：生成 Step 2 所需的 `sampled_TEL.json`（CLM 格式）。

**CLM 格式示例**：
```json
[
  [
    ["My phone number is 1 2 3 4 5 6 7 8 9 0", "1"],
    ["My phone number is 1 2 3 4 5 6 7 8 9 0", "2"],
    ...
  ],
  ...
]
```
每个样本是一个 bag，bag 内为 `[完整文本, 目标token]` 的列表。

**注意**：`mask_text2json.py` 使用 `[MASK]`，为 BERT 格式。若为 LLaMA 准备数据，需要：
- 新建 `mask_text2json_llama.py`，输出上述 CLM 格式，或
- 手动构造符合 `1_calculate_attribution_llama.py` 输入格式的 JSON。

---

## Step 2: 计算归因分数（积分梯度）

**目标**：用积分梯度方法计算每个神经元对隐私 token（如姓名、电话）输出的贡献，是后续聚合与编辑隐私神经元的基础。

**前提**：Step 0 已完成，且已有 `sampled_TEL.json`（或等效 CLM 格式数据）。

```bash
cd /data1/D-PIKE/DEPN-main/src

python 1_calculate_attribution_llama.py \
    --model_name_or_path /path/to/llama3-8B \
    --adapter_dir ../data/model/llama3_8b_ep5_stable_lora4bit \
    --priv_data_path ../data/sampled_TEL.json \
    --output_dir ../results_llama3/ \
    --output_prefix llama3_tel \
    --gpus 0 \
    --max_seq_length 128 \
    --batch_size 16 \
    --num_batch 10
```

**说明**：显存 ≥40GB 时可不加 4-bit；若仅 24GB，需加 `--load_in_4bit` 并适当减小 `--batch_size`、`--num_batch`（如 2、3）或 `--max_seq_length`（如 64）以免 OOM。

**输出**：`../results_llama3/llama3_tel.priv.jsonl`

**验证**：文件存在、无 NaN、梯度分布合理。

---

## Step 3: 神经元过滤

**输入**：Step 2 的 `.priv.jsonl` 文件。

```bash
cd /data1/D-PIKE/DEPN-main/src

python 2_filter_privacy_neurons.py \
    ../results_llama3/ \
    0.01 \
    0.5
```

**输出**：
- `../results_llama3/kn/kn_bag-<rel>.json`
- `../results_llama3/kn/kn_rel-<rel>.json`

其中 `<rel>` 由输入文件名解析：`llama3_tel_ep5.priv.jsonl` → `rel=ep5`，输出为 `kn_bag-ep5.json`。若希望输出为 `kn_bag-llama3_tel.json`，可将 `output_prefix` 设为 `x-llama3_tel`（使 `split('-')[-1]` 为 `llama3_tel`）。

## Step 4: 神经元编辑与评估

```bash

python 3_edit_privacy_neurons_llama.py \
    --model_name_or_path ../data/model/llama3_8b_ep5 \
    --priv_data_path ../data/memorized_TEL.txt \
    --validation_path ../data/enron_data/valid.txt \
    --kn_dir ../results_llama3/kn/kn_bag-ep5.json \
    --gpus 0 \
    --max_seq_length 128 \
    --erase_kn_num 20 \
    --do_random_kn False \
    --input_prefix TEL
```

**输出**：控制台打印编辑前后指标对比。
