# DEPN 项目运行至今所需文件汇总

项目根目录：`/data1/D-PIKE`（遵循仓库项目结构规范）。

> **当前规范路径（本仓库）**  
> | 类型 | 规范路径 |
> |------|----------|
> | 代码 | `srcs/depn/`（子目录：`eval/` 归因/聚合/擦除，`train/` 微调脚本，`data/` 预处理脚本，`utils/` 共用模块，`tools/` 校验脚本） |
> | 数据 | `data/processed/depn/`（DEPN 用到的数据；`data/` 下保留同名符号链接，如 `data/sampled_TEL.json` → `processed/depn/`） |

> | 配置 | `configs/depn/` |

## 一、代码文件

### 1. 核心流程（Llama3 四步）

| 步骤 | 脚本路径 | 作用 |
|------|----------|------|
| Step 2 归因 | **`srcs/depn/eval/1_calculate_attribution_llama.py`** | 对隐私样本算积分梯度，输出 `.priv.jsonl` |
| Step 3 聚合 | **`srcs/depn/eval/2_filter_privacy_neurons_llama.py`** | 从归因结果过滤、聚合神经元，输出 `kn_bag-*.json`、`kn_rel-*.json` |
| Step 4 擦除与评估 | **`srcs/depn/eval/3_edit_privacy_neurons_llama.py`** | 按 kn_bag 擦除神经元，并算暴露度 / PPL / MRR |
| 共用模块 | **`srcs/depn/utils/custom_llama.py`** | Llama 编辑与积分梯度支持，被 Step 2 和 Step 4 引用 |

### 2. 数据与预处理

| 文件路径 | 作用 |
|----------|------|
| **`srcs/depn/train/run_clm_no_trainer.py`** | Llama3 微调（Step 0） |
| **`srcs/depn/data/mask_text2json.py`** | 将采样/掩码文本转为 Step 2 所需的 JSON |
| **`srcs/depn/data/preprocess_enron.py`** | Enron 数据预处理，生成 `temp_data/train.txt`、`valid.txt` 等 |
| **`srcs/depn/data/sample_target_privacy.py`** | 采样目标隐私数据 |
| **`srcs/depn/data/build_privacy_json_tel.py`** | 构建 TEL 隐私 JSON |

### 3. 评估与辅助

| 文件路径 | 作用 |
|----------|------|
| **`srcs/depn/eval/eval_memorized_count_llama_lora.py`** | 记忆化数量评估（NAME/TEL/RANDOM） |
| **`srcs/depn/eval/eval_privacy_metrics_llama_lora.py`** | 隐私指标与质量评估 |
| **`srcs/depn/eval/eval_llama3_finetune.py`** | 微调效果评估 |
| **`srcs/depn/tools/validate_step2_output.py`** | 校验 Step 2 输出 |

### 4. 其他（BERT 或示例）

- `srcs/depn/eval/1_calculate_attribution.py`、`2_filter_privacy_neurons.py`、`3_edit_privacy_neurons.py`：BERT 版流程。
- `srcs/depn/eval/example_usage_llama3.sh`、`2_run_aggregator.sh`、`3_run_editor.sh`：示例/聚合/编辑脚本。

## 三、模型

|------|------|
| **`/data1/D-PIKE/models/llama3-8B/baseline`** 或 **`pretrained_models/llama3-8B`** | 本地基座（Step 2、Step 4 的 `--model_name_or_path`） |

## 四、输出与中间结果（当前 TEL 流程）

### 1. Step 2 输出（归因）

| 路径 | 说明 |
|------|------|
| **`outputs/depn/llama3_tel.priv.jsonl`** | 每行一个 bag 的归因结果，Step 3 的输入 |
| `outputs/depn/llama3_tel.args.json` | Step 2 运行参数 |
| `outputs/depn/llama3_tel.step2_ckpt.*.json` | 归因过程 checkpoint（若启用） |

### 2. Step 3 输出（神经元列表）

| 路径 | 说明 |
|------|------|
| **`outputs/depn/kn/kn_bag-llama3_tel.json`** | 每袋神经元列表，Step 4 的 `--kn_dir` 指向此文件 |
| **`outputs/depn/kn/kn_rel-llama3_tel.json`** | 跨袋聚合后的关系级神经元列表（参考用） |

### 3. 文档与报告（本仓库内）

| 路径 | 说明 |
|------|------|
| `outputs/depn/kn/神经元聚合步骤检测报告.md` | Step 3 四项指标检测说明 |
| `outputs/depn/Step4_擦除结果.md` | Step 4 擦除与 exposure/PPL 记录 |
| `outputs/depn/Step4_擦除关键参数说明.md` | `erase_kn_num`、`kn_dir` 等参数说明 |
| `outputs/depn/擦除与评估检查说明.md` | 暴露度/MRR/困惑度评估正确性检查 |
